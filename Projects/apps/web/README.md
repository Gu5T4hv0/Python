# Frontend (apps/web)

Este diretório deve conter a aplicação Next.js + Tailwind do MicroMentor.

Como inicializar (exemplo):

```powershell
cd apps/web
npx create-next-app@latest . --use-npm
npm install
npm run dev
```

Configurar Tailwind (opcional): siga a documentação oficial do TailwindCSS para Next.js.
